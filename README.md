# 🕹️ Python Console Arcade 🚀

Консольная аркада на Python, в которой собраны классические мини-игры («Камень, ножницы, бумагу» и «Угадай число»). Проект регулярно пополняется новыми играми! 

Игра написана на чистом Python и активно использует функции вместе с их аннотациями для повышения читаемости кода.

---

## 🧠 Функционал и технические особенности проекта:

1. 🧩 **Функции (def и аннотации типов):** код разбит на логические модули (`main_menu`, `rock_paper_scissors`, `guess_the_number`), а использование аннотаций помогает делать код чистым и понятным.
2. 🔄 **Управляющие конструкции и циклы (while, if/elif/else):** организация бесконечных игровых циклов с возможностью выхода и обработки выбора пользователя.
3. 📦 **Структуры данных (Словари и списки):** хранение вариантов выбора и правил игры через словари (`choices`, `rules`), что делает логику расширяемой.
4. 🎲 **Модуль random:** генерация случайных чисел и выбор элементов компьютером для симуляции противника.
5. 🛡️ **Обработка исключений (try-except):** защита программы от некорректного ввода пользователя (защита от падений при вводе букв вместо цифр).
6. 🔤 **Форматирование строк и CLI-интерфейс:** оформление красивого псевдографического меню и аккуратного вывода в терминале.
